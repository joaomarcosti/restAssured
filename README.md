# RestAssured

Um projeto exemplo de utilização de RestAssured.

# Instalaçao e execução

Após cloncar o projeto, abra o projeto e utilize o mvn clean install para instalar as dependencias.

Para executar o projeto, execute a suite.xml

# Ferramentas Utilizadas

- TestNG 7.0.0
- Rest Assured 4.0.0
- Maven 3.1
- Java 1.8
- Faker 1.0.2

# Cenários Avaliados

1. Não deve acessar Api Sem Token (GET)
2. Incluir conta com sucesso (POST)
3. Criar uma conta (POST)
4. Alterar o nome da conta (PUT)
5. Enviar nome de conta já utilizado (POST)
